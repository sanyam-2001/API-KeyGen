API KeyGen
Automate API Keys For your Node Server

Preventing the unauthorized usage of an API is a very cruicial aspect of API development.
Creating and managing your own API Keys can get very labour-intensive and tiresome.
Key Gen generates and maintains your API Keys along with an NPM Module to Authenticate your APIs.
